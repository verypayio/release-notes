Merchant App Release Notes

Version 1.0.0

Release Date: 2026-03-18

New Features

We are excited to announce the launch of version 1.0.0 of the Merchant App! This release includes several new features designed to enhance your experience.

  • Feature 1: Easy setup and configuration process.
  • Feature 2: Enhanced payment processing speed.
  • Feature 3: User-friendly interface with improved navigation.
  • Feature 4: Comprehensive reporting tools.
  • Feature 5: Support for multiple payment gateways.

Improvements

In addition to new features, we have also implemented various improvements:

  • Optimized performance resulting in faster load times.
  • Improved security protocols to safeguard transactions.
  • Minor bug fixes and stability enhancements.

We hope you enjoy using the latest version of the Merchant App. For any questions or feedback, please contact our support team.
